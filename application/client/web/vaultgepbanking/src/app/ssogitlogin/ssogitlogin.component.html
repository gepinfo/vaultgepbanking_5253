Vault directions to add Credentials:

1.Go to Admin Dashboard

2.Click on Vault under System Administration

3.Select Internal App

4.Enter the Token to login to the built-in vault

5.Click on Signin

6.Click on sso/ in the Secret Engines(if you don't find it skip to step 13)

7.Click on login/

8.Click Create secret

9.Enter login/github under Path for this secret

10.Enter the key and the value in the Secret Data section

11.Click Add and do the same for all the key value pairs

12.Click Save

13.Click Enable new engine if there is no sso/ in the Secret Engines

14.Select KV under the Generic category

15.Click Next

16.Enter sso under the Path

17.Click on Method Options

18.Select 1 from the Version dropdown

12.Click Enable Engine

19.Click Create secret

20.Enter login/github under Path for this secret

21.Enter the key and the value in the Secret Data section

22.Click Add and do the same for all the key value pairs

23.Click Save
